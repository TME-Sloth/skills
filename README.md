# TME Sloth Skills

## 安装

在项目或本机环境中，执行：

```bash
npx skills add git@github.com:TME-Sloth/skills.git
```

按 `skills` CLI 的提示完成安装；通常会将本仓库中的 skill 同步到你配置的 skills 目录。

### 环境要求

- 已安装 **Node.js**（建议 ≥ 18），以便使用 `npx`
- 已安装并配置好对应的 **skills** 工具链（由 `npx skills` 提供）
